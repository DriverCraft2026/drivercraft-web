# DriverCraft Web Oficial

Carpeta local fija:
C:\Dev\drivercraft-web

## Probar en local

```powershell
cd C:\Dev\drivercraft-web
npm install
npm run dev
```

Abrir:
http://localhost:3000

## Build de producción

```powershell
cd C:\Dev\drivercraft-web
npm run build
npm run start
```

## Notas
- Enlaces externos abren en nueva pestaña.
- Las imágenes oficiales están en public/images.
- Para cambiar una imagen, reemplaza el archivo con el mismo nombre.
